# GApe

GApe (Get A Page Easily), an interactive design tool for creating one-page content such as a curriculum vitae or leaflets for both personal and commercial use.

## Run

- Before Running

```bash
npm i
```

- Debug Mode

```bash
npm run dev
```

- Build

```bash
npm run build
```
